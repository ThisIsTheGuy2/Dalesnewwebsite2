# Dalesnewwebsite2 — Tailwind landing page scaffold

This repository contains a static Tailwind-based landing page scaffold for Dales Web Solutions.

What I pushed:
- index.html — main page (references assets/logo.svg and assets/hero.svg placeholders)
- css/styles.css — small custom styles
- js/main.js — FAQ accordion and small behaviors
- assets/logo.svg — placeholder logo (replace with your real PNG/SVG if you want)
- assets/hero.svg — placeholder hero image (replace with your supplied hero image file)
- assets/avatar1.svg, avatar2.svg, avatar3.svg — small avatar placeholders
- assets/favicon.svg — tiny favicon placeholder

WhatsApp CTA is wired to: https://wa.me/27774845787 and shows the label "Message Dales Web Solution on WhatsApp".

How to finish:
1. Replace assets/logo.svg with your real logo (you uploaded one in chat) — upload it as `assets/logo.svg` or `assets/logo.png` and update the `index.html` image src if needed.
2. Replace `assets/hero.svg` with your hero image — recommended filename: `assets/hero.jpg` and update the `index.html` src accordingly.
3. Update example domain links and testimonials as desired.

Deploy on GitHub Pages:
1. In the repository Settings -> Pages, set Source to `main` branch and `/ (root)`.
2. Wait ~1 minute and visit: https://thisistheguy2.github.io/Dalesnewwebsite2/ (replace with your username and repo name)
